# 인바디 기반 3D 캐릭터 — Fake Door 사이트 (SWI)

"내 인바디 수치로 3D 캐릭터를 만들고, 운동하는 사람들의 커뮤니티에 참여" 라는 아이디어를
**Fake Door**(가짜 문) 방식으로 시장 검증하는 랜딩 페이지.
방문 → 캐릭터 생성 → 공간 미리보기 → 의견/이메일 까지의 **퍼널 전환**을 측정한다.

- 프론트: HTML / CSS / Vanilla JS + **Three.js**(CDN)로 3D 캐릭터 렌더
- 백엔드: **Google Apps Script + Google Sheets** (이미 배포됨 — 추가 설정 불필요)
- 방문자 추적: **Google Analytics (GA4)**

---

## 1. 실행에 필요한 것 (사전 준비)
- **인터넷 연결** (Three.js / axios를 CDN으로 불러옴)
- **로컬 정적 서버** — ES 모듈/importmap을 쓰므로 `index.html`을 `file://`로 직접 열면 안 됩니다.
  - 권장: VS Code 확장 **Live Server**
  - 또는 이 폴더에서:
    ```bash
    python3 -m http.server 5500     # 또는  npx serve .
    ```
- 백엔드(Apps Script Web App)는 **이미 배포되어 코드에 연결**돼 있습니다(`js/web-api.js`). 평가자가 별도로 만들 필요 없습니다.

## 2. 실행 방법 (단계별)
1. 이 폴더를 로컬 서버로 열고 `index.html` 접속 (`http://localhost:5500/`)
2. **Hero** → "캐릭터 만들어볼래요?" 클릭
3. **성별 선택**
4. **인바디 입력**
   - 필수: 체중 / 체지방률 / 골격근량
   - 선택: "부위별 근육량 직접 입력" 체크 시 팔·몸통·다리 입력 (정밀 체형)
   - "내 캐릭터 생성하기"
5. **결과 화면**: 인바디 비율이 반영된 3D 캐릭터가 단계별로 생성됨 → "춤 보여주기"로 애니메이션 확인
6. "**친구들과의 공간으로 가기**" → 내 캐릭터로 입장한 3D 공간 미리보기(드래그로 둘러보기)
7. "**친구를 초대해서…**" → **의견/이메일 입력** → 제출 → Thank You

## 3. 수집되는 데이터 (퍼널)
각 방문은 세션 1행으로 Google Sheet(`sessions` 탭)에 누적됩니다.

| 컬럼 | 기록 시점 |
|---|---|
| `id`, `timestamp` | 페이지 방문 |
| `character` = O | "캐릭터 생성"(폼 제출) |
| `community` = O | "친구들과의 공간으로 가기" |
| `final` = O | "친구를 초대해서…" 클릭 |
| `email`, `feedback` | 의견 제출 |

→ 방문 → 생성 → 공간 → 의견 **전환율**을 한 시트에서 계산할 수 있음. (방문자 수는 GA4에서 별도 확인)

## 4. 파일 구조
```
index.html              진입점 (5+섹션 단일 페이지)
css/web.css
js/
  web-main.js           섹션 전환 / 폼 / 퍼널 로깅
  web-api.js            Apps Script 통신 (upsert)
  web-character.js      인바디 → 본 스케일 + 캐릭터 생성/춤
  character.js          Three.js 캐릭터 로드/애니메이션/바닥고정
  web-coming-soon.js    의견 섹션 배경 캐릭터
  web-space.js          "친구들과의 공간" 3D 목업
asset/                  캐릭터(남/여) GLB + 미리보기
template/               디자인 템플릿(Bootstrap 등)
```

## 5. 참고 / 문제 해결
- `file://`로 열면 모듈 로드 실패 → **반드시 로컬 서버**로 여세요.
- 제출 시 콘솔에 **빨간 CORS 에러**가 보일 수 있는데 **정상**입니다. GET 요청은 서버까지 도달해 시트에는 정상 저장되고, 응답만 못 읽는 것뿐입니다.
- 캐릭터 GLB(수 MB) 로딩에 잠시 시간이 걸릴 수 있습니다.
- 배포 URL: (배포한 주소가 있으면 여기에 기입)
